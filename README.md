# flosnidor9.github.io — Personal Archive

개인 자료를 보관하고, 감상하고, 전시하기 위한 공간입니다.

---

## 용도

이 사이트는 외부에 공개하기 위한 포트폴리오가 아닙니다.
수집한 이미지, 기록, 생각의 조각들을 **폴더(카테고리) 단위로 분류하고 시각적으로 아카이빙**하는 개인 전시 공간입니다.

- **보관** — 사진, 이미지, 메모 등 개인 자료를 구조적으로 저장
- **감상** — 저장된 자료를 몰입감 있는 UI로 편안하게 감상
- **전시** — 폴더별로 큐레이션된 형태로 자료를 배열하고 탐색

---

## 구조

```
/                     # 메인 랜딩 (인터랙션 히어로)
  └── /[folder]       # 폴더별 게시물 리스트
        └── /[post]   # 개별 자료 상세 페이지
```

---

## 디자인 컨셉

**Liquid Glass Archive** — 리퀴드 글래스(Liquid Glass)와 그레인 블러(Grain Blur)를 결합한 몰입형 반응형 웹사이트.

- 메인 화면은 `favorites` 태그 이미지 중 1장을 무작위로 불러와 전체 화면을 채웁니다.
- 마우스(또는 터치)로 블러와 그레인을 걷어내며 이미지를 탐색합니다.
- 스크롤하면 이미지가 줌아웃되며 리퀴드 그라데이션 배경으로 전환됩니다.
- 폴더들은 글래스모피즘 카드 형태로 가로 캐러셀에 나열됩니다.
